Chris Davie Data engineer.

I’m a Python developer and Data Engineer with a PhD in computational Physics from Imperial College. Since graduating 2 years ago, I’ve designed and helped implement 3 significant software projects in Python. I’m building fundamental computing skills and enjoy solving problems with a mathematical focus.

I’ve built the batch processing pipelines for Speechmatics, rebuilt the structure of the Machine Learning libraries and built multiple models using these. I also have experience of numerical Python development, including Numpy, Scipy, Matplotlib and Cython, developing novel algorithms to solve fantasy sports, mapping and fluid dynamics problems. I also have some experience building algorithms in C++.

My portfolio describes the projects I’ve worked on in more detail.

I’m automating building speech to text models using Speechmatics’ machine learning technologies. I redesign and re-implement legacy and proof of concept code, automating it, making it more robust, maintainable and faster. I’m also gathering the data and designed and built the storage and post-processing.

I designed the pipeline and database, and we then built now run the automated pipeline. So far around 20 languages have been completed, including German, Spanish, Italian, Portuguese, French, Polish, Swedish, Romanian and Russian.

Building on this success, I’ve recently redesigned the structure of the underlying machine learning code to increase the performance, stability and portability, which has been implemented. We’re now building a high quality Japanese model.

Driven by a colleague and myself, our team has recently introduced testing, including unit, component, regression and integration tests, continuous integration and code review. The automation pipeline is in Python using the Luigi batch processing framework, the database is built using SQLAlchemy and PostgreSQL and the scrapers use Scrapy.
